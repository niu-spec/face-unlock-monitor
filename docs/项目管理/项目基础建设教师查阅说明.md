## 七、建议核验顺序（约 5～10 分钟）

1. Branches / Network / Contributors  
2. 打开本目录「Git 昵称」或根目录 `CONTRIBUTORS.md`  
3. `openspec/changes/archive/` 任选一条  
4. 打开 `/api/docs/`（或附图 / `?format=openapi`）  
5. GitHub Actions 最近运行；可选 Jenkins `:8080`
